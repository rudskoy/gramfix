Gramfix Grammar Fix - simple and secure clipboard history manager with a macOS 26+ liquid glass interface parts.

   Works on local llms on M CPU family of macbooks; depends on MLX/ollama inference.

   Features:
      MLX Inference Based Models (Downloaded Automatically): No cloud for AI required
      Persisted clipboard (In Application Support).
      Encrypted by default clipboard History.
      Shortcuts (Paste, Paste Without Spoiling System Clipboard, Favourites)
      Can Describe Pictures (Made Just For Fun)
      Copied preserving and displaying all the formatting (generated paste results are cleaned from formatting).
      Automatically Detects Useful Clipboard Items (Copied At Least Once For More Than 3 Unique Days)

Made for fun, 90% vibecoded via Cursor/Claude Code.
