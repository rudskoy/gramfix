# Gramfix

Simple and secure clipboard history manager with a macOS 26+ liquid glass interface.

![Gramfix Screenshot](https://i.imgur.com/VfpBnNd.png)

## Features

- **Encrypted by default** - Clipboard history stored securely
- **Local AI processing** - MLX/Ollama inference, no cloud required
- **Smart clipboard** - Preserves formatting, detects useful items
- **Keyboard shortcuts** - Quick paste, paste without spoiling clipboard, favorites
- **Image descriptions** - AI-powered image analysis (just for fun)
- **Persistent storage** - History saved in Application Support
- **Auto-detection** - Identifies frequently used clipboard items

## Requirements

- macOS 26.0+ (Tahoe)
- Apple Silicon (M-series) MacBook for M processor.
- `xattr -d com.apple.quarantine /Applications/Gramfix.app` (until author has apple account).
